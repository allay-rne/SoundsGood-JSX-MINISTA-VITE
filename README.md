# SoundsGood — Стриминг сервис 🚀

**Многостраничный стриминг-сервис (музыка/книги)**  
*Вёрстка с нуля • JSX + SCSS + Vite + Minista + БЭМ*  
Практический проект по курсу Friendly Frontend (Александр Ламков).

## ✨ **Что реализовано (7 страниц)**
- ✅ **6/7 страниц**: Главная, Каталог, Профиль, Карточка, Книги, ~~Загрузка музыки~~
- ✅ JSX компоненты + семантика HTML5
- ✅ Адаптив (Mobile-First, Grid + Flexbox)
- ✅ Слайдеры + Бургер-меню
- ✅ БЭМ методология (Block__Element--Modifier)
- ✅ Vite + Minista сборка (HMR + продакшен)

## 🛠 **Технологии**

JSX - Vite 7 - Minista - SCSS - JavaScript - БЭМ

## 🚀 **Быстрый старт**
```
# Клонируй
git clone https://github.com/allay-rne/SoundsGood-JSX-MINISTA-VITE

# Установи
npm install

# Запуск
npm start   
npm run build # Продакшен
```

##🔧 Скрипты
```
npm start     # ⚡ Dev server (HMR)
npm run build # 🔨 Production build
```

##📂 Структура
```
src/
├── assets/
│   ├── images/     # Оптимизированные
│   ├── icons/      # SVG спрайт
│   └── fonts/      # Web fonts
├── components/     # Button/, Card/ (атомы + index.jsx)
├── constants/      # Данные страниц
├── layouts/        # Header, Footer (общие)
├── modules/        # Слайдеры, бургер JS
├── pages/          # 6/7 страниц
├── sections/       # Hero, Features
├── utils/          # Хелперы
├── global.jsx      # Глобальные стили
└── main.js         # Entry point
```
